Memory Mapper Test Program
-------------------------------------------------------------------------------

各スロットに装着されている MemoryMapper対応RAM の容量を調べるツールです。
DOS2 または NEXTOR のコマンドプロンプトから、下記の要領でスロット番号を指定して下さい。

A> MAPCHK

これで全スロットの MemoryMapper対応RAM を調べます。

A> MAPCHK 83

これで、指定のスロットの MemoryMapper対応RAM だけを調べます。
83 は、16進数表記のスロット番号です。ENASLT で Aレジスタに指定する値ですね。
